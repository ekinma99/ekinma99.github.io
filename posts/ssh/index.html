Github SSH Key

March 24, 2021

Open SSH will Soon be Depreciated

GitHub is depreciating its use of Open SSH to authenticate users when pushing changes to GitHub. Thus, in this post I will show you how to add an SSH key to GitHub. I am under the assumption that the audience primarily reading this article has some experience with using GitHub and Git Bash (for Windows) or Terminal (for Linux/Mac). With that being said let’s get started!

Entering Commands in Bash

Open up Git Bash (for Windows) or Terminal (for Linux/Mac).

You can do this in any folder but I decided to do this in my desktop. If you would like to change directories, do so now.

Next you want to type the following command,

ssh-keygen -t rsa -b 4096 -C "youremail@domain.com" 
# here you want to enter your email for GitHub in the quotations

A key pair will now be generated. You will then be asked to enter where you want this key pair to be saved. You can change the path if you like or keep it in the directory you are currently in.

Next, you will be prompted to enter a passphrase. The passphrase is a password that can be different from your GitHub password. It will not be shown on the command line.

Hit enter, once you finished entering the passphrase.

Once the passphrase has been confirmed, a public key will be generated and saved into the designated folder.

The key’s fingerart and randomart will also be generated.

Enter the following command,

eval $(ssh-agent -s)

An Agent pid will be generated following the command.

Enter the following command,

ssh-add ~/.ssh/id_rsa

You will be prompted to enter the passphrase you created before. Next you will need to copy the key.

There are 2 options:

  1. Youc can go to the id_rsa.pub file and copy its contents OR
  2. You can enter the following command to copy the key,
clip < path_of_the_id_rsa.pub_file # you would enter the path of the id_rsa.pub file after the less than symbol.

Switching over to GitHub

  1. Click on your profile picture.
  2. Click on Settings.
  3. Under Personal Settings, click on SSH and GPG keys.
  4. Click the button for New SSH Key.
  5. Give the SSH key a title. You can title it as you please; I titled mine’s as Home Desktop.
  6. Then, you want to paste the SSH key into the key textbox.
  7. Last step, click add SSH key once it’s pasted.

You have now successfully added the SSH key to GitHub. Happy coding!
